# Desplega Homebrew Tap

Official Homebrew tap for Desplega tools.

## Installation

```bash
brew tap desplega-ai/tap
```

## Available Formulas

| Formula | Description |
|---------|-------------|
| `file-review` | File review tool with CodeMirror editor and Vim bindings |

## Quick Install

```bash
brew install desplega-ai/tap/file-review
```

## Updating

```bash
brew update
brew upgrade file-review
```
